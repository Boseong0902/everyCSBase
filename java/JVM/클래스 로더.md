# 클래스 로더

클래스 로더는 `.class` 파일을 찾아서 JVM의 Runtime Data Area에 적재하는 역할을 한다.

## 역할

- 바이트코드 파일을 읽어온다.
- 클래스 파일이 올바른지 검증한다.
- 클래스가 실행될 준비를 한다.
- 초기화 단계를 수행한다.
- 결과물을 Runtime Data Area의 Method Area에 저장한다.

클래스는 프로그램 시작 시점에 전부 올라가는 것이 아니라, 필요해지는 시점에 동적으로 로딩될 수 있다.
