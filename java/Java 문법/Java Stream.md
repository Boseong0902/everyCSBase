# Java Stream

Stream은 컬렉션이나 배열 같은 데이터 소스를 선언적으로 처리하기 위한 API다.

## Stream과 Collection의 차이

### Stream

- 반복을 내부적으로 수행한다.
- 병렬 처리가 가능하다.
- 반복 순서를 효율적으로 내부에서 조정할 수 있다.

### Collection

- 모든 값을 메모리에 저장해둔다.
- 컬렉션에 추가하기 전에 값 계산이 완료되어 있어야 한다.
- 반복 순서는 개발자가 외부 반복문으로 지정한다.
- 기본적으로 병렬 처리를 직접 지원하지 않는다.

## 핵심 차이

Collection은 데이터를 보관하는 자료구조에 가깝고, Stream은 데이터를 어떻게 처리할지를 표현하는 처리 파이프라인에 가깝다.
